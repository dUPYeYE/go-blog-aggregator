# GO Blog Aggregator
This is a simple blog aggregator for the Go programming language. It is a work in progress and is not yet ready for use.

## Prerequisites
- Go 1.23 or later
- PostgreSQL

## Installation
```bash
go install github.com/dUPYeYE/go-blog-aggregator
```

## Usage
```bash
gator help
```
